Станок ЧПУ для производства балясин

Данный станок ЧПУ представляет собой механизм для движения токарного резца, либо шпинделя с фрезой

по заданному профилю.

Подходит для любого токарного станка.

Длина обрабатываемого профиля до одного метра.

Станок является исполнительным механизмом для изготовления симметричных деталей из дерева.

Это могут быть балясины, ножки для мебели и другие детали.

Для изготовления балясины достаточно загрузить профиль детали в программу, задать режим и запустить

изготовление балясины на исполнение. Станок ЧПУ сам оптимально просчитает количество проходов по заготовке.

Продажа технологии станка для производства балясин обговаривается.

Станок для производства балясин, цена 50т рублей. Продаем станки дешево.
